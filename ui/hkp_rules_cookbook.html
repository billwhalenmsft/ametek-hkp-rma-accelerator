Confidence adjustments:

  • Tier 1 OEM in-warranty + clear defect → +0.05 (boost up to max 0.97)
  • Repeat claim from same OEM for same SKU within 60 days → −0.10 (force human review)
  • Confidence below 0.70 → humanReviewRequired = true (route through Manager Approval)

Cost-tier guardrails:

  • approve_replace AND unit cost > tier auto-approve cap → require Manager Approval
  • approve_refund AND amount > $5,000 → require Executive Approval
